Welcome to NADRE tutorial's documentation!
==========================================

.. image:: figures/logo-nadre-noname.jpg
   :alt: NADRE Project
   :scale: 50 %
   :align: right

This tutorial shows you how to programmatically interact with the Invenio_ based `NADRE Repository <http://nadre.ethernet.edu.et/>`_.

.. _Invenio: https://invenio-software.org/

The tutorial cosists of the following two sections:

- the `first <search-engine-api.html>`_ one shows some examples on how to search object stored in the digital repository building HTML query;
- the `second <submit-records.html>`_ one show how programmatically submit records to the repository.

The presentation and the video recording of this tutorial are available on NADRE repository and have the following DOIs:

- Presentation `10.20372/ethernet:1533718023.48 <http://dx.doi.org/10.20372/ethernet:1533718023.48>`_
- Video tutorial `10.20372/ethernet:1533719889.38 <http://dx.doi.org/10.20372/ethernet:1533719889.38>`_
